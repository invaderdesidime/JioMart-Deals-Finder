Steps -

- Download Zip File
- Unzip file
- Open Google Chrome or Any Chromium-based browser and go to Manage Extensions
- Turn ON developer mode
- Click on Load unpacked button and select the unziped file
- Now make sure to PIN the chrome extension so that it is visible at all times
- Go to JioMart and search for anything
- Click on JioMart extension to see the products sorted by discount
- You can also open any category page and do the same.
- Extension Supports - JioMart Search and Category Pages.
